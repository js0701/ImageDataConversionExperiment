Picutre Processing

A sample about fetch pixel data from pictures, swap the RGBA value and show it on Canvas. It shows the process can be accelerated by use of ImageData conversion extension. The CapturePicture button use the extension to encode ImageData to picture and render it on canvas.

WebGL Texture Upload

A sample about use HTMLImage as textures in WebGL. It shows the process can be accelerated by use ImageData Conversion extension.
